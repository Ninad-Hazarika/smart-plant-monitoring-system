# 🌱 Smart Plant Monitoring System - CAD Design

This folder contains the complete CAD design and enclosure development for the Smart Plant Monitoring System created using Onshape.

---

## 📌 Design Objectives

- Create a compact smart planter enclosure
- Integrate electronics safely inside the body
- Design dedicated water tank placement
- Create proper routing space for wiring
- Ensure easy accessibility for maintenance
- Build an aesthetic and functional product design

---

## 🛠 Designed Components

### 🌿 Plant Holder
- Dedicated space for soil and plant placement
- Stable and compact structure

### 💧 Water Tank
- Separate compartment for water storage
- Optimized for mini water pump integration

### ⚡ Electronics Section
- Placement for:
  - ESP32
  - Relay Module
  - Soil Moisture Sensor
  - DHT11 Sensor
  - OLED Display
  - Wiring Management

### 🔩 Enclosure Design
- Compact all-in-one body
- Practical layout for real-world usage

---
