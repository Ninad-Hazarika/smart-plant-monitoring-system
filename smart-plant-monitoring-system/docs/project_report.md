# Project Report: Smart Plant Monitoring System

## Introduction

This project is an ESP32-based smart plant monitoring and irrigation system. It reads soil moisture, temperature, and humidity, displays values on an OLED screen, and controls a water pump through a relay.

## Objective

The goal is to create a low-cost plant care system that can help monitor plant health and automate watering.

## Methodology

1. Connect sensors to ESP32.
2. Read sensor values using Arduino IDE.
3. Display readings on OLED.
4. Control pump using relay.
5. Add Wi-Fi web interface for phone-based control.
6. Design schematic and PCB using EasyEDA.

## Result

The system successfully reads sensor values and allows pump control through a local web page.

## Conclusion

This project helped demonstrate the use of ESP32, sensors, relay modules, OLED display, and basic IoT control.
