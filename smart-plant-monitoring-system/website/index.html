Smart Plant Monitoring System

ESP32 based plant monitoring and irrigation project

About

This project monitors soil moisture, temperature, and humidity and controls a water pump using ESP32.

Features

  • Soil moisture monitoring
  • DHT11 temperature and humidity sensing
  • OLED display
  • Wi-Fi pump control
  • Relay based irrigation
